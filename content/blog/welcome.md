This is your first blog post. It's a placeholder. Delete it and write your own.

Posts live in `content/blog/` as Markdown files. Each post has a frontmatter block at the top with the title, date, optional cover image, optional excerpt, and a draft flag. The post body is plain Markdown.

If you add a `cover` image to the frontmatter, the post card on the blog index will render with a cover. Without one, the card uses a text-only layout. Same component, two looks.

When you're ready, delete this file (or change `draft` to `true` to hide it without deleting). The blog index page will auto-hide entirely if there are no published posts.
